Changing color of text

This is all inline

style="color:red" There are some default colors, but most often people are using hex codes

Defaults: https://www.w3schools.com/colors/colors_names.asp

These go within the hierarchy

Can also do style="color:hex code"

Can also use rgb style="color:rgb(0,255,0)"

not in line

put name of tag in beginning, open with curly bracket, name of what you wwant to change, then what you're going to change it to, then end with semicolon

web safe fonts: https://www.w3schools.com/cssref/css_websafe_fonts.asp

Best practice to not do in-line styling because it's so inconvenient. Put it all in one place at the beginning (internal).

Important to make color scheme before hand so it's not atrocious. Knowing styling beforehand is very important. Style with a destination vs Styling on the fly

Making things consistent overall helps things look better

Bring over what was in style into style.css

link style sheets in style.css

make sure all css is in the same place. Never have style in multiple places.
