Part 2: Web Design 101

color

online always using RGB, which is additive color mixing

subtractive color mixing, cmyk, is for ink

always use RGB for web so it matches

RGB optimized for screens 0 to 255

CMYK optimized for print

hex value is another way of representing RGB

all three colors in full in additive will get white

styling

trending colors come from brands

the bigger the organization, the longer it takes for them to catch up to the trends

color theory

building a color scheme

some colors work well together

look for inspiration for color palletes from artists or nature

monochromatic

easiest color pallete, hard to mess up

start with neutrals then go to different shades

would work well for a clothing website since you're very image focused

facebook good example of monochromatic because everything is blue, because content is the focus

complementary

hardest to pull off

across each other on the color wheel

low saturation dark colors to give warm feeling, but high contrast

analogous
