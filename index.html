Tips for navigating github

  1. Atom -- Make any changes
  2. GitHub Desktop -- fetch from origin to make sure your local repository will match the origin repository
  3. GitHub Desktop -- provide a Summary of changes you made (the commit message)
  4. GitHub Desktop -- Commit to main
  5. GitHub Desktop -- Push origin
  6. GitHub Web -- visit your URL and you should see updates
